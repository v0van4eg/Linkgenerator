Написать веб приложение на Python. Хостинг картинок для вставки ссылок в документы.
Пользователь загружает через форму картинки. Файлы сохраняются в каталог с именем клиента, который указывается на
странице. Так же указывается имя товара. Генерируются прямые ссылки для каждого файла и выводятся в правой
части страницы в столбик. Перед ссылкой сделать превью 64px.
Сгенерированные ссылки можно копировать в буфер обмена.
Так же ссылки можно выгрузить в txt-файл по одной ссылке в каждой строке.
Каталог с картинками обслуживается веб-сервером Nginx.


Написать функции генерации XLSX-документов из шаблона. Для каждого клиента уже готов свой шаблон.

Для Мегамаркет megamarket.xslx
Столбцы:
"Код товара СММ(обязательно)" "Ссылка на основное фото"	"Ссылка на доп. фото №1" "Ссылка на доп. фото №2" - "Ссылка на доп. фото №9"

Для ЭЛИЗЕ elise.xlsx
Столбцы:
"Бренд"	"Номенклатура" "Артикул" "Базовый Штрихкод" "Измерение" "ЗначениеИзмерения (Если косметика то колористика)"	"Заголовок (Без бренда)" "КраткоеОписание" "СпособПрименения" "ЛинейкаБренда" "Каждый цвет (ПО мише еще и по объёмам)" "ИмяФайлаКартинки1-"ИмяФайлаКартинки20"
в ячейках "ИмяФайлаКартинки№" ссылки по одной в каждой. Число ячеек по количеству картинок.
